InfraWatchX is a modular AI-based classroom monitoring system designed to assist educational institutions in:

Automating attendance

Analyzing student engagement (focus & emotions)

Detecting safety-related events (crowding, fights, prohibited objects)

Unlike single-model or rule-based solutions, InfraWatchX follows a multi-model decision-fusion architecture, where each AI model is responsible for a specific task and their outputs are aggregated by an Administrative Decision Engine.

This project focuses on AI system engineering, scalability, and real-world deployability, making it suitable for final-year academic evaluation and industry demonstration.
